# 開發計畫與需求決策紀錄

> 本檔記錄與業主（品保）確認過的所有需求決策，為開發依據。

## 系統定位

- **品保部門專用**的每日環境 5S/GMP 巡檢工具，取代紙本「衛生檢查紀錄表 S12501F」。
- **不做**多巡檢類型通用引擎（HACCP/工安/消防/設備另開專案，可參考本架構）。
- 巡檢項目仍**不 hard code**：29 項與三大類放 DB 設定表，由 JSON seed 管理，可增修不動程式。

## 已確認的關鍵決策

| 主題 | 決策 |
|---|---|
| 判定用語 | 沿用紙本：合格V / 不合格X / 待處理△ / 復驗O。「未改善/改善中/已改善」是缺失層的獨立狀態 |
| 日期 | 內部 ISO 儲存，PDF 輸出民國年（對齊紙本） |
| 權責單位 | 主檔管理、**可跨部門複選**（多對多）。名單：原料班/果醬調理班/果餡調理班/果醬(餡)充填班/花生班/包裝班/資材/工務課/工務-廢水場/配料室（待補） |
| 檢查人員 | 主檔管理：林俊廷/王品儒/李育儒（待補），巡檢表可複選 |
| 結構化資料 | Supabase PostgreSQL（免費方案） |
| 檔案儲存 | **熱/冷雙層**（2026-07-11 定案）：熱層 Supabase Storage（App 即時讀寫）；冷層 Google Drive 鏡像歸檔（P2 以 OAuth 個人授權接上，操作者可直接在共用資料夾瀏覽/下載）。緣由：Google 服務帳號無法寫入個人 Gmail Drive（無儲存配額之硬限制） |
| 空間管理 | Supabase 免費 1GB；照片壓縮後約 200-300KB/張。Drive 鏡像完成後，已結案且逾 N 個月之舊照自動自熱層清除（冷層永存），空間不爆 |
| Drive 結構 | `Photos/`（照片）、`Reports/Daily|Monthly|Annual`（PDF）、`Data/`（JSONL + Excel 匯出） |
| 修改/刪除 | 允許修正記錯：軟刪除 + 稽核軌跡；**已改善（結案）缺失鎖定**不可改刪 |
| 品保 SLA | 缺失開立後 **5 個工作天**內追蹤改善完成（跳過週末+國定假日），計入品保 KPI |
| 逾期提醒 | **LINE 為主**（Email 不做）；即將逾期 + 已逾期提示；發送對象與內容可選 |
| 自助回報 | 可將單一單位的缺失以專屬安全連結發給該單位，對方免登入回報改善與照片 |
| 照片 | 上傳前瀏覽器端自動壓縮；多張、拖曳排序、刪除、更換、後補；改善前/後分類 |
| 自動儲存 | 所有輸入即時存檔，不設儲存鈕 |
| 部署 | Vercel（免費），GitHub push 自動部署 |
| PWA 離線 | 納入（工廠收訊不良場景），本機草稿 + 恢復連線同步 |

## 分階段

### P0 架構奠基（本階段）
- [x] Next.js + TS + Tailwind 專案骨架（Clean Architecture 目錄）
- [x] DB Schema migration（RLS / 軟刪除 / 已結案鎖定 / 工作天函式 / 稽核表）
- [x] Seed：紙本 29 項、權責單位、人員、2026 假日、SLA 設定
- [x] Domain 層：實體、Repository 介面、StorageProvider 介面
- [x] 文件：架構 / 資料庫 / 本計畫
- [ ] 使用者執行 migration + seed（Supabase SQL Editor）

### P1 巡檢閉環（可取代紙本）
- 今日巡檢表（29 項快速判定 UI、自動儲存）
- 不合格/待處理 → 展開缺失（說明/建議/**權責單位複選**/期限預設5工作天/拍照）
- 照片壓縮 + Google Drive 上傳（服務帳號）
- 未改善缺失 / 已改善缺失 / 歷史巡檢 三頁
- 每日 PDF（1:1 紙本：檢查表 / 狀況說明含未結案舊缺失 / 照片 2×3 含編號日期）
- Email 登入（Supabase Auth）

### P2 儀表板與分析
- 首頁 Dashboard（今日/本月統計、快捷）
- 績效分析頁：單位異常排行、狀態占比、MoM/YoY、趨勢、Pareto、重複缺失、熱點矩陣
- 品保 KPI：追蹤達成率、平均追蹤天數、逾期件數（幫品保提升績效）
- 即將逾期/已逾期清單與標示
- 月報 PDF、Data/ 資料紀錄檔（JSONL + Excel）自動產出
- （參考使用者提供的品質單位 5S 月報再優化）

### P3 年報與閉環強化
- 年度報告 PDF（12個月趨勢、YoY、單位排名、Top 10 缺失）
- LINE 通知（即將逾期/已逾期/手動發送，對象與內容可選）
- 自助回報連結（單位免登入回報 + 改善後照片 → QA 覆核結案）
- 電子簽核（廠長/衛生管理人員/檢查人員）
